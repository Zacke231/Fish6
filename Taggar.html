Vad är HTML och attrubuter?

HTML

HTML står för Hypertext Markup Language och är standardmärkningsspråket för att skapa webbsidor. Det består av en serie element som används för att strukturera innehåll på internet.

Vad är head-tagen och h1 samt p och a taggarna?

Fil-taggar

head-taggen: Existerar för att innehålla metainformation om dokumentet, såsom dess titel och länkar till skript och stilmallar.

h1-taggen används för att definiera den viktigaste rubriken på en webbsida, och det finns sex nivåer av rubriktaggar, h1 till h6, där h1 har högst prioritet.

p-taggen används för att definiera ett stycke text och till sist a-taggen används för att skapa hyperlänkar.

Vad är style-tagen?

Style i HTML

style-taggen användss för att inkludera CSS (Cascading Style Sheets) direkt inuti ett HTML-dokument.

Vad är body-tagen?

Body för kod

body-taggen innehåller allt innehåll på en webbsida som ska visas för användaren, såsom till exempel text och bilder

Vad är span-tagen?

Span för formatering

span-taggen används för att gruppera eller formatera en del av en text inuti ett dokument ungefär, men utan att ge den någon extra betydelse.

Vad är div-tagen?

Div för gruppering

div-taggen i HTML står för division eller avsnitt och används för att gruppera blockelement för att formatera dem med CSS. Till skillnad från span-taggen, som är inline och används för att gruppera inline-element, är div en blocknivå-tagg och påverkar därför layouten genom att skapa ett “block”.

Vad är img-tagen?

Bild eller ingen bild

img-taggen: Används för att bädda in bilder i ett HTML-dokument. Den har attribut som src som anger bildens URL och alt som ger en textbeskrivning av bilden för skärmläsare och när bilden inte kan visas så står det istälet.
