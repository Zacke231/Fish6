Vad är Color attributen i css?

Bra med färg

Color-attributet används för att ange textfärgen för ett element så som för den här texten. Du kan använda fördefinierade färgnamn, hexadecimala värden, RGB, RGBA, HSL, HSLA, och även currentColor som refererar till elementets nuvarande textfärg

Vad är border attributen i css?

Trevligt med ram

Border-attributet är en samlingsattribut som definierar bredden, stilen och färgen på en elements ram. Det kan inkludera värden som solid, dotted, dashed, och andra för att ange ramens stil

Vad är @media tagen i css?

Trevligt att kunna se på mindre skärm

@media-taggen är en regel som används i mediafrågor för att tillämpa olika stilar beroende på enhetens typ eller skärmens egenskaper, som bredd och höjd. Detta är användbart för att skapa responsiv design

Varför man inte ska använda vw

Använd inte

Att använda vw (viewport width) kan vara problematiskt eftersom det inte tar hänsyn till användarens zoomnivå eller andra visningsinställningar som kan påverka läsbarheten. Det kan leda till designproblem, särskilt på mobila enheter där skärmstorleken är begränsad och användaren kan behöva zooma in för att läsa texten bekvämt. Dessutom kan användning av vw för textstorlek orsaka problem med text som inte skalas korrekt och blir för stor eller för liten jämfört med resten av elementen på sidan.
