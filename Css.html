Vad är Css?

Css för stil

CSS, som står för Cascading Style Sheets, är ett stilmallsspråk som används för att beskriva presentationen av ett dokument skrivet i HTML

CSS används för att definiera hur HTML-element ska visas på skärmen

Varför behöver vi Css?

Css för bra design

Vi behöver CSS för att det ger oss möjligheten att separera innehållet från designen på en webbsida. Detta gör det enklare att underhåla och uppdatera webbsidans utseende utan att ändra dess innehål. Dessutom kan CSS förbättra webbsidans laddningstider eftersom stilmallarna kan cachas av webbläsaren

Vad är alt-attributet för bilder?

Text istället för bild

Alt-attributet för bilder är en textbeskrivning som används för att förklara innehållet och syftet med en bild på en webbsida. Detta är särskilt viktigt för användare som använder skärmläsare.

Vilken fördel finns det med att ha css filen separat?

Alltid bra

Att ha CSS-filen separat från HTML-dokumentet har flera fördelar så som:

Det blir lättare att underhålla och uppdatera webbsidans design.
Man kan återanvända samma CSS-fil på flera sidor, vilket ger en enhetlig stil över hela webbplatsen.

Det hjälper också med struktur då separationen skapar en tydlig struktur i koden, vilket gör det enklare för andra utvecklare att förstå och arbeta med koden.
